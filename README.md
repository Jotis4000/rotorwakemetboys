# rotorwakemetboys

by Panagiotis Sachinis, Barnabas Szeibert, Tibbe Blaauboer
March 2026

Full Blade Element Momentum model capable of calculating thrust, torque, power, and relevant coefficients for a finite propeller.
In-built optimization script using Differential Evolution and Bezier curves to calculate optimal propeller geometry for given operating condition.
Corrections and assumptions implemented can be found in the associated report.
